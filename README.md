# Космическая бургерная
![img](./src/images/logo.jpg)

Stellar burger - веб приложение космической бургерной. Является полноценным фронтенд приложением-магазином, с возможностью регистрации, создания заказа и просмотра как всех заказов, так и каждого заказа по отдельности. Так же есть возможность редактировать свои данные в личном кабинете пользователя.

Приложение развернуто на [GitHub Pages](https://kozhemyakinaelizaveta.github.io/react-burger/index.html).


Учебный проект на курсе [react-разработчик](https://practicum.yandex.ru/react/) от yandex. Сделан по [макету](https://www.figma.com/file/zFGN2O5xktHl9VmoOieq5E/React-_-Проектные-задачи_external_link?node-id=0%3A1&t=L1Qw4Ncywvdufa0e-0).
```
git clone https://github.com/KozhemyakinaElizaveta/react-burger.git
cd react-burger
npm ci
npm start
```
##### Стек технологий
- HTML
- React
- TypeScript
- JavaScript
- Redux
- Figma
- Jest и Cypress (использованы для тестирования)
